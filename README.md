Sports Predictions App 📱
A React Native mobile app that provides live sports data, match predictions, and social features like chat and friends system.

🚀 Features
- Live sports data via external API
- Match predictions system
- Real-time chat between users
- Friends / social system
- User profiles
- Push notifications for match updates
- Authentication (Google login)

 🧠 Tech Stack
-  React Native (Expo)
- Supabase (Auth, Database, Realtime)
- REST API integration (sports data)
- Push Notifications
- Google OAuth

⚙️ How it works

- Users sign in via Google authentication
- App fetches live match data from external API
- Predictions are stored and shared in real-time
- Chat system uses Supabase realtime channels
- Notifications are triggered for match updates
